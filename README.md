# MyScript
自用的python小脚本

## check_url_survival.py
多线程批量检测URL存活率

usage: python3 check_url_survival.py your_urls.txt

![avatar](https://github.com/sakura404x/MyScript/blob/main/photos/Xnip2021-02-14_11-36-39.jpg)

默认会输出两个文件，存活的url会输出至表格，检测失败的URL会输出至错误日志
